== Description ==

<Strong>Contact Form 7 Signature Addon</Strong> is a free WordPress plugin. 

In this Digital World are some businesses a digital signature functionality is considered useful, especially when you want extra security. Multiple signature pads can be added to Contact Form 7 using the digital signature feature.

You need Contact Form 7 installed. it make simple way to setup in contact form 7 as per another field we setup in contact form 7 that way it be.

This plugin Make <Strong>Signature field in contact form 7</Strong>.

<h3>FEATURES:</h3>

<ul>
<li>Required field</li>
<li>Customize signature pad background and pen color</li>
<li>Customize signature pad width and height</li>
<li>Customize signature pen width</li>
<li>Draw and clear your signature</li>
<li>Change signature clear button text</li>
<li>Send signature in email</li>
<li>Setting id and class of the signature field (optional)</li>
</ul>
